## 配置要求

### 飞书应用配置
1. 在飞书开放平台创建自建应用
2. 配置机器人和事件订阅
3. 获取并配置以下参数：
   - LARK_APP_ID
   - LARK_APP_SECRET
   - LARK_ENCRYPT_KEY
   - LARK_VERIFICATION_TOKEN

### 环境变量配置
```bash
# 应用配置
APP_NAME=PM Digital Employee
APP_ENV=production
APP_PORT=8000

# 数据库配置
DATABASE_URL=postgresql+asyncpg://...

# 飞书配置
LARK_APP_ID=...
LARK_APP_SECRET=...
LARK_ENCRYPT_KEY=...
LARK_VERIFICATION_TOKEN=...

# LLM配置（如需要）
LLM_API_KEY=...
```
