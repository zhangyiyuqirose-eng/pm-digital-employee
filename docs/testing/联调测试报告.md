### 9.1 问题诊断

| 问题 | 原因 | 影响 |
|------|------|------|
| PM机器人重复发送消息 | internal_process_message缺少幂等检查 | 用户收到多条重复回复 |

### 9.2 修复方案

```python
# app/api/lark_webhook.py
@internal_router.post("/process-message")
async def internal_process_message(request: Request):
    # 添加幂等检查
    idempotency_service = get_message_idempotency_service()
    if not await idempotency_service.check_message(msg_id):
        logger.info("Message already processed, skipping")
        return {"code": 0, "msg": "ok", "result": {"skipped": True}}
```
