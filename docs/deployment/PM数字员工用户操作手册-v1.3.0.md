## 1. 系统概述

### 1.1 什么是PM数字员工？

PM数字员工是一款基于飞书的智能项目管理助手，专门服务于项目管理部门的日常工作。通过自然语言对话，帮助项目经理完成项目查询、周报生成、任务更新、风险预警、成本监控等工作，显著提升工作效率。

### 1.2 核心特性

| 特性 | 说明 |
|------|------|
| 🎯 **唯一入口** | 飞书作为唯一用户交互入口，无需安装额外应用 |
| 🤖 **智能对话** | 自然语言交互，无需记忆复杂命令 |
| 📊 **12大功能** | 覆盖项目管理全流程的核心功能 |
| 🔒 **安全合规** | 项目级数据隔离，权限管控 |
| 📈 **多源录入** | 飞书卡片、Excel、在线表格、文档解析四种录入方式 |

### 1.3 适用人群

- 项目经理（PM）
- 项目管理办公室（PMO）
- 技术负责人
- 部门管理人员

## 2. 快速入门

### 2.1 如何访问？

**步骤一：打开飞书**
- 在电脑或手机上打开飞书应用

**步骤二：找到PM数字员工**
- 在飞书通讯录中搜索"PM数字员工"
- 或加入项目管理群，群内有PM数字员工机器人

**步骤三：开始对话**
- 发送任意消息即可开始交互
- 建议先发送"你好"测试连接

### 2.2 对话基本规则

| 规则 | 说明 |
|------|------|
| 自然语言 | 无需严格命令格式，系统智能识别意图 |
| 项目上下文 | 在项目群中对话时，自动关联当前项目 |
| 多轮交互 | 复杂操作可能需要多轮对话补充信息 |
| 卡片确认 | 重要操作会发送卡片供确认 |

### 2.3 快速测试命令

```
你好                    → 查看系统介绍
查看项目列表            → 查看您可见的项目
项目总览 [项目名]       → 查看某个项目的整体情况
帮助                    → 获取功能帮助
```

PM数字员工：
根据《项目管理规范（2024版）》第三章规定，项目立项流程如下：

   - PMO评估可行性

2. **立项审批**
   - 提交立项申请
   - 技术委员会评审
   - 分管领导审批

3. **立项备案**
   - PMO登记立项信息
   - 分配项目编号
   - 创建项目档案

**参考来源：**
1. 项目管理规范（2024版）第三章
2. 立项审批流程指南

*本回答基于制度文档生成，如有疑问请咨询PMO*
```

**步骤二：查询具体规定**
```
用户：变更管理规定是什么？
PM数字员工：
根据《变更管理规范》相关条款：

**变更类型：**
1. 需求变更：功能需求、业务规则的修改
2. 设计变更：技术方案、架构设计的修改
3. 资源变更：人员、预算、时间的调整

**变更流程：**
1. 提交变更申请（填写变更申请单）
2. 影响评估（项目经理评估影响范围）
3. 变更审批（根据变更等级分级审批）
4. 变更执行（按审批结果执行变更）
5. 变更记录（更新项目文档和档案）

**参考来源：**
1. 变更管理规范 第四章

*本回答基于制度文档生成，如有疑问请咨询PMO*

## 5. 常见问题解答

### 5.1 基础问题

**Q1：如何开始使用？**
A：打开飞书，搜索"PM数字员工"，发送消息即可开始对话。

**Q2：必须要绑定项目才能使用吗？**
A：在项目群中使用时自动关联项目；私聊时需要指定项目名称或编号。

**Q3：如何查看我有哪些项目权限？**
A：发送"查看项目列表"查看您可见的项目。

### 5.2 功能问题

**Q4：周报生成后可以修改吗？**
A：周报生成后发送文本内容，您可以复制修改，系统不强制入库。

**Q5：任务进度更新有误怎么办？**
A：重新发送更新指令修正，如"更新进度 订单模块 70%"。

**Q6：制度问答的答案不准确？**
A：系统基于知识库检索，如有疑问请咨询PMO确认。

**Q7：文档解析失败怎么办？**
A：检查文档格式是否正确，文件大小不超过50MB，必要时重新发送。

### 5.3 数据格式问题

| 字段类型 | 格式要求 | 示例 |
|----------|----------|------|
| 日期 | YYYY-MM-DD | 2024-04-19 |
| 日期时间 | YYYY-MM-DD HH:MM | 2024-04-19 14:30 |
| 进度 | 0-100整数 | 80 |
| 金额 | 数字，可带小数 | 100000.00 |
| 状态 | 下拉选项 | 进行中/已完成 |

### 5.4 权限问题

**Q8：看不到某个项目？**
A：需要项目管理员授权，请联系管理员添加查看权限。

**Q9：无法更新任务进度？**
A：检查是否有该项目的编辑权限。
