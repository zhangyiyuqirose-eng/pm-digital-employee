# 项目经理数字员工系统 - 用户手册

## 一、系统简介

项目经理数字员工系统是一个智能化项目管理助手，通过飞书为您提供项目管理相关服务，帮助您提高工作效率、规范管理流程、及时发现风险。

### 核心功能
- **项目总览**：快速查看项目状态、进度、里程碑等信息
- **周报生成**：自动生成项目周报，支持导出Word文档
- **WBS生成**：自动生成项目工作分解结构
- **任务管理**：更新任务进度，查看任务状态
- **风险预警**：自动识别风险，及时预警提醒
- **成本监控**：查看项目成本执行情况
- **制度答疑**：解答项目管理制度相关问题
- **会议纪要**：自动生成会议纪要
- **合规初审**：预立项/立项材料合规检查

## 二、使用方式

### 1. 单聊使用

在飞书中找到"项目经理数字员工"机器人，直接发送消息即可：

```
@项目经理数字员工 查看项目001的状态
```

### 2. 群聊使用

在绑定了项目的飞书群中，发送消息即可：

```
生成周报
```

## 三、功能详解

### 1. 项目总览

**使用场景**
快速了解项目的整体状态，包括进度、里程碑、任务统计等。

**操作方式**
```
查看项目状态
项目概览
```

**返回内容**
- 项目基本信息（名称、状态、开始结束日期）
- 整体进度百分比
- 里程碑列表及状态
- 任务完成统计
- 当前风险概览

### 2. 周报生成

**使用场景**
每周自动生成项目周报，支持导出Word文档。

**操作方式**
```
生成周报
本周报告
导出周报
```

**返回内容**
- 本周工作完成情况
- 进行中任务列表
- 风险提示
- 下周工作计划
- 支持导出Word文档

### 3. WBS生成

**使用场景**
根据项目信息自动生成工作分解结构（WBS）。

**操作方式**
```
生成WBS
工作分解结构
```

**返回内容**
- 项目分解层级结构
- 各工作包明细
- 责任人分配
- 支持导出Word文档

### 4. 任务管理

**使用场景**
更新任务进度，查看任务状态。

**更新任务进度**
```
更新任务 任务001 进度80%
任务001 完成度80%
```

**查看任务列表**
```
查看任务
任务列表
```

**返回内容**
- 任务ID和名称
- 任务状态和进度
- 负责人信息
- 截止日期

### 5. 风险预警

**使用场景**
查看项目风险列表，了解风险等级和处理状态。

**操作方式**
```
查看风险
风险列表
```

**返回内容**
- 风险等级（高/中/低）
- 风险描述
- 处理状态
- 负责人

### 6. 成本监控

**使用场景**
查看项目成本执行情况，对比预算与实际支出。

**操作方式**
```
查看成本
成本情况
预算执行
```

**返回内容**
- 预算总额及分类
- 实际支出总额及分类
- 预算执行率
- 剩余预算

### 7. 制度答疑

**使用场景**
询问项目管理制度、规范要求相关问题。

**操作示例**
```
项目周报的提交时间是什么？
立项材料需要哪些内容？
里程碑变更流程是什么？
```

**返回内容**
- 制度规定解答
- 规范要求说明
- 参考来源引用

### 8. 会议纪要

**使用场景**
根据会议内容自动生成会议纪要。

**操作方式**
```
生成会议纪要
会议内容：[会议记录内容]
```

**返回内容**
- 会议基本信息
- 会议内容摘要
- 决议事项
- 行动项
- 支持导出Word文档

### 9. 合规初审

**使用场景**
预立项/立项材料的合规性初审。

**操作方式**
```
检查立项材料
材料合规审查
```

**返回内容**
- 完整性检查结果
- 合规性检查结果
- 问题清单
- 修改建议

## 四、交互卡片说明

系统会通过交互卡片与您进行交互，包括以下类型：

### 1. 项目总览卡片
显示项目整体信息，包含：
- 项目状态概览
- 进度条
- 里程碑时间线
- 快捷操作按钮

### 2. 风险预警卡片
显示风险详情，包含：
- 风险等级标识
- 风险描述
- 处理建议
- 处理按钮

### 3. 周报展示卡片
显示周报内容，包含：
- 周报摘要
- 导出按钮
- 编辑按钮

### 4. 澄清确认卡片
用于多轮补参，包含：
- 参数输入框
- 确认按钮
- 取消按钮

### 5. 任务更新卡片
显示任务详情，包含：
- 任务信息
- 进度更新按钮
- 状态切换按钮

### 6. 审批状态卡片
显示审批流程，包含：
- 审批节点
- 当前状态
- 操作按钮

## 五、常见问题

### Q1：如何绑定项目群？
联系项目管理员，将飞书群与项目进行绑定。

### Q2：为什么看不到某个项目的数据？
需要确保您有该项目的访问权限，请联系项目管理员添加权限。

### Q3：周报生成不准确怎么办？
系统根据项目任务数据生成周报，请确保任务数据录入准确完整。

### Q4：制度问答没有找到答案？
系统基于知识库回答问题，如果找不到答案，请联系管理员更新知识库。

### Q5：生成的文档格式有问题？
导出的Word文档可能存在格式差异，可以手动调整后使用。

## 六、使用技巧

### 1. 简化指令
使用简短的指令可以更快触发功能：
```
周报 → 生成周报
风险 → 查看风险
成本 → 查看成本
```

### 2. 指定项目
在单聊中可以指定项目：
```
项目001的周报
```

### 3. 批量操作
某些操作支持批量：
```
更新任务 任务001、任务002 进度100%
```

### 4. 导出文档
生成报告后可以导出Word文档：
```
导出周报
导出WBS
```
