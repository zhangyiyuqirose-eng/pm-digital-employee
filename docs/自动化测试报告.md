## 四、新增模块详情

### 4.1 数据加密模块（encryption.py）

**功能**:
- `DataEncryptor`: AES-128对称加密（Fernet）
- `MaskUtils`: 数据脱敏工具（手机号、邮箱、身份证等）
- 符合等保三级要求

**API**:
```python
from app.core.encryption import DataEncryptor, MaskUtils

# 加密
encryptor = DataEncryptor()
encrypted = encryptor.encrypt_phone("13812345678")
decrypted = encryptor.decrypt_phone(encrypted)

# 脱敏
masked = MaskUtils.mask_phone("13812345678")  # "138****5678"
```

### 4.2 日志脱敏模块（log_sanitizer.py）

**功能**:
- `LogSanitizer`: 自动脱敏日志内容
- `SanitizedLogger`: 脱敏日志器包装
- 支持手机号、邮箱、身份证、Token等脱敏

**API**:
```python
from app.core.log_sanitizer import get_sanitized_logger

logger = get_sanitized_logger(__name__)
logger.info("用户手机号13812345678登录")  # 自动脱敏
```
